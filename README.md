# 🐍 Python Mini Projects

## 📌 Description

This repository contains a collection of small Python projects developed as part of my learning journey in programming.

These mini projects focus on improving problem-solving skills, understanding core programming concepts, and applying Python in different scenarios such as games, data processing, and basic security tools.

---

## 🎯 Objectives

* Practice Python fundamentals
* Improve algorithmic thinking
* Explore real-world use cases
* Build small but functional applications

---

## 📂 Included Projects

### 🎮 Tic Tac Toe

A simple terminal-based game that allows two players to play Tic Tac Toe.

### 🔐 Security Manager

A basic tool for handling passwords and simple security operations such as generation and storage.

### 📊 Data Analytics

Scripts for analyzing datasets, generating statistics, and visualizing data.

> More mini projects will be added over time.
